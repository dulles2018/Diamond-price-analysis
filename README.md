# Diamond-price-analysis
An R project analyzing the factors that influence diamond prices through data visualization, statistical analysis, and linear regression. 
# Diamond Price Analysis in R

## Project Overview

This project explores how diamond characteristics such as carat, cut, color, and clarity influence price using the built-in `diamonds` dataset from the `ggplot2` package.

## Objectives

- Explore the dataset
- Visualize price distributions
- Compare prices by cut, color, and clarity
- Analyze correlations
- Build a linear regression model

## Dataset

- Source: `ggplot2::diamonds`
- Observations: 53,940
- Variables: 10

## Tools

- R
- ggplot2
- dplyr
- GGally
- corrplot

## Project Structure

- `Diamond_Price_Analysis.R` – R analysis script
- `Diamond_Price_Analysis.Rmd` – R Markdown report
